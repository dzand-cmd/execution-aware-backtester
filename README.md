# Execution-Aware Backtesting Framework

**Author:** dzand-cmd  
**Project Type:** Quantitative Research / Algorithmic Trading  
**Language:** Python  
**Status:** In Progress  

---

## Overview

Execution-aware backtester for evaluating trading strategies with realistic assumptions:

- Executes trades at bid/ask prices
- Models slippage and latency
- Compares mid-price PnL vs executable PnL

Designed as a portfolio project to demonstrate:

- Quantitative research skills
- PnL attribution and alpha degradation analysis
- Python data analysis and visualization
